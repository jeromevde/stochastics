For GBM with constant $\sigma$, E[exp(−$\tfrac{1}{2}\sigma^2$t + $\sigma$B(t))] equals:
For X ~ N(0, $\sigma^2$t): E[e^X] = e^{$\sigma^2$t/2}. So E[e^{−$\tfrac{1}{2}\sigma^2$t + $\sigma$B(t)}] = e^{−$\tfrac{1}{2}\sigma^2t} \cdot$ e^{$\tfrac{1}{2}\sigma^2$t} = 1. This is the exponential martingale property.
