The characteristic function of a Poisson process $N_\lambda(1)$ is:
Definitions: The characteristic function of a random variable $X$ is $\Phi_X(\theta) = E[e^{i\theta X}]$, the expected value of $e^{i\theta X}$. For a Poisson random variable with parameter $\lambda$, we compute this expectation using the Poisson PMF.

Reasoning: For $N_\lambda(1) \sim \text{Poisson}(\lambda)$: $\Phi_{N_\lambda(1)}(\theta) = E[e^{i\theta N_\lambda(1)}] = \sum_{n=0}^{\infty} e^{i\theta n} P(N_\lambda(1) = n) = \sum_{n=0}^{\infty} e^{i\theta n} \frac{\lambda^n e^{-\lambda}}{n!} = e^{-\lambda} \sum_{n=0}^{\infty} \frac{(\lambda e^{i\theta})^n}{n!} = e^{-\lambda} e^{\lambda e^{i\theta}} = e^{\lambda(e^{i\theta} - 1)}$. This function uniquely characterizes the Poisson distribution.

Answer: $e^{\lambda(e^{i\theta} - 1)}$.
