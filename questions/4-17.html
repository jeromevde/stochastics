For GBM with constant $\sigma$, E[exp(−$\tfrac{1}{2}\sigma^2$t + $\sigma$B(t))] equals:
Definitions: For a normal random variable $X \sim N(\mu, \sigma^2)$, the moment generating function is $E[e^X] = e^{\mu + \sigma^2/2}$. Here, we have $-\frac{\sigma^2}{2}t + \sigma B(t)$ where $B(t) \sim N(0, t)$. Reasoning: Let $Y = -\frac{\sigma^2}{2}t + \sigma B(t)$. Then $Y \sim N(-\frac{\sigma^2}{2}t, \sigma^2 t)$. Using the MGF formula: $E[e^Y] = \exp(-\frac{\sigma^2}{2}t + \frac{\sigma^2 t}{2}) = e^0 = 1$. This is the exponential martingale property: the process $\exp(-\frac{\sigma^2}{2}t + \sigma B(t))$ is a martingale with expectation 1, crucial for Girsanov's theorem and GBM. Answer: The expectation equals 1.
