A Poisson process $N_\lambda(t)$ has increments that follow a:
Definitions: A Poisson process $N_\lambda(t)$ is a counting process that models the number of events occurring by time $t$, with arrival rate $\lambda$ (expected number of events per unit time). It has independent, stationary increments. Reasoning: For a Poisson process, the increment $N_\lambda(t) - N_\lambda(s)$ represents the number of events in $(s, t]$. This follows a Poisson distribution with parameter $\lambda(t-s)$: $$P(N_\lambda(t) - N_\lambda(s) = k) = \frac{[\lambda(t-s)]^k e^{-\lambda(t-s)}}{k!}$$ The mean and variance both equal $\lambda(t-s)$, and the distribution depends only on the interval length $(t-s)$, not the specific times. Answer: Poisson distribution with parameter $\lambda t$.
