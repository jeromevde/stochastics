A compound Poisson process $X(t) = \sum_{i=1}^{N(t)} J_i$ differs from a Poisson process in that:
Definitions: A compound Poisson process is $X(t) = \sum_{i=1}^{N(t)} J_i$, where $N(t)$ is a Poisson process counting the number of jumps, and $J_i$ are i.i.d. random variables (jump sizes) drawn from some distribution, independent of $N(t)$. Reasoning: In a standard Poisson process $N(t)$, each jump has size 1, so $N(t)$ counts discrete events. In a compound Poisson process, each event $i$ has a random size $J_i$ that can vary. For example, $J_i$ could be normally distributed (for stock price log-returns) or deterministic. The process still has finite jump activity (finitely many jumps in any finite interval) but allows heterogeneous jump sizes. Answer: Jump sizes can vary (according to a jump size distribution $J$).
