If dx = −$\kappa$x dt + $\sigma$ dB and f(x,t) = xe^{$\kappa$t}, then by Itô's lemma df equals:
Definitions: Given an SDE $dx = -\kappa x \, dt + \sigma \, dB$ and a function $f(x,t) = xe^{\kappa t}$, we apply Itô's lemma: $df = f_t \, dt + f_x \, dx + \frac{1}{2}f_{xx}(dx)^2$.

Reasoning: Compute partial derivatives: $f_t = \kappa x e^{\kappa t}$, $f_x = e^{\kappa t}$, $f_{xx} = 0$. Substitute: $df = \kappa x e^{\kappa t} \, dt + e^{\kappa t}(-\kappa x \, dt + \sigma \, dB) = \kappa x e^{\kappa t} \, dt - \kappa x e^{\kappa t} \, dt + \sigma e^{\kappa t} \, dB = \sigma e^{\kappa t} \, dB$. The drift terms cancel perfectly, leaving only a stochastic integral. This transformation is key to solving the Vasicek SDE.

Answer: $df = \sigma e^{\kappa t} \, dB$ (drift terms cancel).
