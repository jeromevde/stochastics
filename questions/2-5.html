A random variable X on ($\Omega, \mathcal{F}$, P) is a measurable function from:
Definitions: A measurable function $f: (\Omega_1, \mathcal{F}_1) \to (\Omega_2, \mathcal{F}_2)$ is a function such that for every $B \in \mathcal{F}_2$, the preimage $f^{-1}(B) = \{\omega \in \Omega_1 : f(\omega) \in B\} \in \mathcal{F}_1$. A random variable is a measurable function from a probability space to the reals.

Reasoning: A random variable $X$ assigns a real number to each outcome $\omega \in \Omega$. Mathematically, $X: \Omega \to \mathbb{R}$. For measurability, we need $X^{-1}(B) \in \mathcal{F}$ for all Borel sets $B \in \mathcal{B}(\mathbb{R})$. This makes $X$ a function from $(\Omega, \mathcal{F})$ to $(\mathbb{R}, \mathcal{B}(\mathbb{R}))$.

Answer: A random variable is a measurable function from $(\Omega, \mathcal{F})$ to $(\mathbb{R}, \mathcal{B}(\mathbb{R}))$.
