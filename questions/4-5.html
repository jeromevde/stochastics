In GBM, the log return ln(S(t)/S(0)) has distribution:
Definitions: For GBM with $S(t) = S(0) \exp\left((\mu - \frac{\sigma^2}{2})t + \sigma B(t)\right)$, the log return is $\ln(S(t)/S(0))$. Since $B(t) \sim N(0, t)$, this is a linear transformation of a normal random variable.

Reasoning: From the GBM solution: $\ln(S(t)/S(0)) = (\mu - \frac{\sigma^2}{2})t + \sigma B(t)$. Since $B(t) \sim N(0, t)$, we have $\sigma B(t) \sim N(0, \sigma^2 t)$. Adding the deterministic drift term: $\ln(S(t)/S(0)) \sim N\left((\mu - \frac{\sigma^2}{2})t, \sigma^2 t\right)$. Note that the price $S(t)$ itself has a lognormal distribution, but the log return is normal.

Answer: The log return has distribution $N\left((\mu - \frac{\sigma^2}{2})t, \sigma^2 t\right)$.
