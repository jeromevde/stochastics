Definitions: The Variance Gamma (VG) process is a pure-jump LÃ©vy process constructed by subordinating (time-changing) Brownian motion with drift using an independent Gamma process. It provides a flexible model for asset returns with heavier tails than Gaussian. Reasoning: The VG process is defined as $X(t) = \theta \Gamma_\nu(t) + \sigma B(\Gamma_\nu(t))$, where $B$ is standard Brownian motion, $\Gamma_\nu$ is an independent Gamma process (the "stochastic clock"), $\theta$ is the drift, and $\sigma$ controls diffusion intensity. The Gamma subordinator $\Gamma_\nu(t)$ represents random time: the process "runs" at different speeds. This construction generates infinite jump activity and can match empirical features like skewness and excess kurtosis. Answer: Time-changing Brownian motion with drift by an independent Gamma process. Practical Use Case: The Variance Gamma model is widely used for pricing equity options because it generates heavy-tailed, skewed return distributions matching empirical data - something Black-Scholes cannot achieve. By time-changing Brownian motion with a Gamma subordinator, VG produces realistic volatility smiles across strikes. Practitioners calibrate VG to liquid vanilla options and then use the calibrated model to price exotic payoffs (e.g., barriers, digitals) consistently. The model's tractability via characteristic functions makes it competitive with stochastic volatility models while avoiding complex Monte Carlo simulations.
